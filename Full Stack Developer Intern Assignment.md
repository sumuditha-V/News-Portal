## **Full Stack Developer Intern - Technical Assessment** 

## **Objective** 

Build a simple News Portal application using .NET Core, React, and MS SQL Server. 

Duration: 6 - 8 Hours 

## **Functional Requirements** 

## **1. User Authentication** 

Create a login functionality using user data stored in a database. 

Requirements: 

- Login page 

- Validate user credentials 

- After successful login, allow access to the application 

- Logout functionality 

## **2. News Listing Page** 

Display a list of news articles fetched from a public API. 

Requirements: 

- Show news title 

- Show news image (if available) 

- Show short description 

- Show source name 

- Show published date 

- Display data in a clean list or card layout 

## **3. News Details Page** 

When a user clicks on a news item: 

Display a detailed view including: 

- Title 

- Full description 

- Image 

- Source 

- Published date 

## **Technical Requirements** 

## **Backend (.NET Core Web API)** 

- Implement user authentication 

- Create secured APIs for frontend consumption 

- Handle API integration logic if needed 

## **Frontend (React)** 

- Use functional components 

- Use React Hooks (useState, useEffect) 

- Implement routing (List page + Details page) 

- Handle loading and error states 

- Store authentication state 

## **Database (MS SQL Server)** 

- Store user credentials 

- Validate login from database 

## **Public News API** 

Use the following free public API (no authentication required): 

https://saurav.tech/NewsAPI 

Available endpoints include multiple news categories and country-based feeds. 

## **Bonus Tasks**

- Show loading spinner while fetching data

- Clean and reusable components
